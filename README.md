## command line options

All options are optional, the default input and output are stdin and stdout.

* -i <input file>
* -o <output file>
* -d (yydebug)
* -t (lexer tokens)
* -s (symbol table)

## Implementation

* **Lexical analyzer**
* **Sytax analyzer**
* **Semantic analyzer**
* **Intermediate code generator**
* **Code optimizer**
* **Target code generator**
